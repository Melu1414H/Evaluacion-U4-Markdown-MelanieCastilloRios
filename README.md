# Este repositorio muestra el proceso para configurar y publicar un repositorio en GitHub.
## Proceso de configuración

1. Inicie sesión en GitHub web.
2. Inicie sesión en GitHub Desktop.
3. Cree un repositorio en GitHub Desktop.
4. Enlace ese repositorio con GitHub web.
5. Selecione "Settings" en GitHub web.
6. Presione en "Danger Zone" la opción de "Change Visibility".
7. Recargue las dos páginas (GitHub web y GitHub Desktop).
8. Finalmente verifiqué si ya estaba publica hasta que en el repositorio apareciera "Public".
